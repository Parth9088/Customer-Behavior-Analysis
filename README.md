# 📌 Overview

The objective of this project is to analyze customer shopping behavior and identify patterns in customer demographics, purchasing habits, product preferences, and subscription trends to support data-driven business decisions.

The dataset contains **3,900 customer transactions** across multiple product categories and includes customer demographics, purchase information, discounts, subscriptions, and review ratings.

---

# 📁 Dataset

- **Records:** 3,900 customer transactions
- **Features:** 18 columns
- **Format:** CSV
- **Data Includes:**
  - Customer Demographics
  - Product Information
  - Purchase Details
  - Subscription Status
  - Discounts & Promotions
  - Shipping Information
  - Customer Reviews

# 🔄 Project Steps

### 1️⃣ Data Loading
- Imported dataset using Pandas.
- Performed initial data inspection and validation.

### 2️⃣ Exploratory Data Analysis (EDA)
- Analyzed customer demographics and purchase trends.
- Generated summary statistics and visualizations.

### 3️⃣ Data Cleaning
- Handled missing values.
- Standardized column names.
- Removed inconsistencies and duplicates.

### 4️⃣ SQL Analysis
- Loaded cleaned data into PostgreSQL.
- Executed business queries to extract insights.
- Performed aggregations, filtering, and segmentation.

### 5️⃣ Dashboard Development
- Built an interactive Power BI dashboard.
- Added KPIs, charts, filters, and slicers.

### 6️⃣ Reporting & Presentation
- Created a business report summarizing findings.
- Developed a presentation using Gamma.

# 📈 Results & Insights

- Male customers generated higher revenue than female customers.
- Young adults contributed the highest overall revenue.
- Express shipping customers spent more on average.
- Loyal customers represented the largest customer segment.
- Several products showed high dependency on discounts.
- Subscription adoption remains a major growth opportunity.

# 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/yourusername/customer-shopping-behavior-analysis.git
```

### Install Required Libraries

```bash
pip install pandas numpy matplotlib sqlalchemy psycopg2
```

### Run Jupyter Notebook

```bash
jupyter notebook
```

### Execute SQL Queries

Import the cleaned dataset into PostgreSQL and execute the SQL queries provided in the repository.

### Open Power BI Dashboard

Open the `.pbix` file using Power BI Desktop to explore the dashboard.
